Answers

1 Which color will be used to display the paragraph text?
Answer: Orange.

2 Explain why – briefly describe which rule takes precedence and why.
The rule .izjava { color: orange !important; } takes precedence because it includes the !important declaration. Even though the ID selector #obvestilo is normally more specific, any !important rule overrides non-important ones.

3 What color would the text be if the element did not have the class izjava?
Answer: Blue. Without the class izjava, the !important rule is removed. Among the remaining selectors, the ID selector #obvestilo has the highest specificity, so the color from that rule (blue) would apply.
